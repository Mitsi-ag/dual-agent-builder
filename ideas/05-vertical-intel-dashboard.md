## The Product
Industry-specific real-time intelligence dashboard. Template: news aggregation + AI synthesis + alerting for ONE vertical.

### V1 Vertical Options
1. **Supply Chain** — port delays, commodity prices, logistics disruptions
2. **Crypto/DeFi** — protocol launches, TVL changes, regulatory moves
3. **Real Estate** — market data, policy changes, auction results (AU-specific via PropTrack)
4. **AI/ML** — model releases, company pivots, funding rounds, open source trends

### V1 Features
- 50+ curated sources per vertical (RSS, APIs, scrapers)
- AI synthesis: daily brief, trend detection, anomaly alerts
- Custom alert rules (price thresholds, keyword triggers, sentiment shifts)
- Team dashboard with shared annotations
- API for integrating into existing workflows
